Drag & drop images here or click to browse — PNG, JPEG, ICO — multiple files supported
⚠ ICO layer limit: The .ICO format’s directory uses a single byte for width & height, so it only supports layers up to 256×256. Sizes above 256 px (among currently selected sizes: ) will still generate PNG + individual .ICO files, but are excluded from the combined Multi-Size .ICO to keep it standards-compliant. Including them would create duplicate 0×0 directory entries that most decoders can’t distinguish.
