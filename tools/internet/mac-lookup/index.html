MAC Address Lookup

Look up the vendor/manufacturer from a MAC address OUI.

⚠️ Local backend offline — run python app.py in the python/ folder.

You can enter a full MAC or just the first 3 octets (OUI). Separators (colon, dash, dot, none) all accepted.
