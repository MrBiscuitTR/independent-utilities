Port Checker

Check TCP/UDP ports on any public host. Leave host blank to use your own IP.

⚠️ Local backend offline — run python app.py in the python/ folder.

Up to 30 ports. Comma/space separated. Append /udp for UDP (e.g. 51820/udp). TCP is default.
